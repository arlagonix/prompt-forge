Height Parameter (Textarea only)

The height parameter controls the number of rows displayed in a textarea. Default is 4 rows.

{{short | height=3}} {{medium | height=8}} {{long | height=20}} {{description | type=textarea | label=Details | height=12 | default=Enter long text...}}
